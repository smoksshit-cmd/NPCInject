# 👤 NPC Project Inject

> *Тихушки, которые делают ваш RP живым.*

Расширение для **SillyTavern**, которое **случайно вписывает NPC в ваш чат** — как случайных прохожих, фоновых персонажей или внезапных знакомых. Они не ломают сюжет, они его **обогащают хаосом** 💫

---

## 🐾 Что это вообще такое?

Представь: ты сидишь в таверне, общаешься с персонажем, и тут мимо проходит какой-то странный тип, бормочет что-то себе под нос, бросает взгляд и уходит. Или к барной стойке подсаживается незнакомка и делает замечание про погоду.

**Вот это оно.** Расширение само подкидывает случайных NPC в сцену — мягко, ненавязчиво, с учётом того, что сейчас происходит.

---


## 🎲 Три режима работы

| Режим | Как работает |
|-------|-------------|
| 🎲 **Полный рандом** | Раз в N сообщений система кидает кубик и вписывает кого-то в нарратив |
| 🏷 **Тематический** | Анализирует сцену (таверна, рынок, бой, вечеринка) и подбирает подходящего NPC |
| ✋ **Только /spawn** | Автоматика выключена, NPC появляются только по команде |

---

## 👫 Динамические группы

NPC может прийти **не один**! С настраиваемым шансом (слайдер «Шанс группы») система автоматически подтянет 1-2 случайных попутчиков из пула.

Пример: ты вызвал Марко, а с ним пришла ещё Лина — просто потому что они «встретились по дороге». У каждого свой голос и характер.

---

## 🛡️ Защита от поломки сюжета

- NPC появляется **мягко** — «у барной стойки кто-то знакомый по голосу», а не «ВНИМАНИЕ ВСЕМ»
- Если сцена серьёзная (теги: `death`, `trauma`, `finale`) — инжекция **пропускается**
- NPC **не доминирует** и **не хайджекает** нарратив — он фоновый, живой, но ненавязчивый

## ✏️ Создание своих NPC

Нажми **➕ Создать NPC** в панели расширения и заполни:

| Поле | Что писать |
|------|-----------|
| **Имя** | Как зовут персонажа |
| **Возраст** | Например: 25 лет |
| **Рост** | Например: 170 см |
| **Внешность** | Волосы, глаза, особенности |
| **Одежда** | Что обычно носит |
| **Характер** | Какой он/она по натуре |
| **Фразы** | Примеры стиля речи (через `;`) — AI не копирует их дословно, а ориентируется на тон |
| **Особенности** | Уникальные черты, привычки, секреты |
| **Теги сеттинга** | `modern`, `fantasy`, `sci-fi`, `any` |
| **Локации** | Где может появиться: `tavern`, `street`, `market`... |

### Пример NPC:

```
Имя: саша
Возраст: 28
Рост: 190 см
Внешность: красивая
Одежда: чистая
Характер: есть
Фразы: "я саша"
Особенности: солнечный
Теги: modern, any
Локации: bar, cafe, street
```

## 👥 Спутники (группы)

У каждого NPC можно добавить **спутников** — они появятся вместе как группа.

Для каждого спутника указывается:
- Имя
- **Связь с лидером** (подруга, коллега, случайная знакомая...)
- Внешность, характер, стиль речи

Пример: Саша + его напарница Лина (давняя подруга) — оба со своими репликами и характером.

## 📥 Импорт / Экспорт

- **📥 Импорт JSON** — загрузи файл с NPC (один объект или массив)
- **🔄 Сбросить к дефолтным** — вернуть стандартный набор тихушек

### Формат JSON для импорта:

```json
{
  "id": "marco_01",
  "name": "Саша",
  "age": "28",
  "height": "190",
  "appearance": "красива",
  "wearing": "чистаа",
  "personality": "есть",
  "catchphrases": ["я саша"],
  "special": "солнечный",
  "tags": ["modern", "any"],
  "setting": ["bar", "cafe", "street"],
  "enabled": true
}
```

## ⚙️ Настройки

| Настройка | Что делает |
|-----------|-----------|
| **Каждые N сообщений** | Как часто проверять — «пора ли кому-то появиться?» |
| **Шанс (%)** | Вероятность появления NPC при проверке |
| **Шанс группы (%)** | Вероятность что NPC придёт с 1-2 попутчиками |
| **Авто-определение сцены** | Система сама определяет локацию по контексту чата |
| **Сцена вручную** | Указать локацию руками (таверна, улица...) |

## 🔌 Свой AI эндпоинт (опционально)

Можно подключить **отдельный эндпоинт** для генерации NPC, чтобы не тратить токены основного чата:

1. Включи тумблер «AI эндпоинт»
2. Введи URL (например `https://api.openai.com/v1`)
3. Введи API ключ
4. Нажми 🔄 чтобы загрузить модели
5. Выбери модель

## 🎭 Роли NPC

При появлении NPC получает **случайную роль** из пула:

- 🚶 Случайный прохожий
- 👋 Старый знакомый главного героя
- 🕵️ Информатор / свидетель
- 🍜 Тот, кто просто сидит в углу и ест
- ⚔️ Внезапный союзник / враг
- 🌀 Фоновый хаос-агент

---

## 💖 Фишечки

- 👤 Иконка рядом с именем персонажа показывает, что в этом ответе был NPC
- NPC **реагируют на контекст** — они читают последние 4 сообщения и отвечают уместно
- Фразы — это **ориентир тона**, а не скрипт. AI импровизирует в стиле персонажа
- Всё хранится в `localStorage` — ничего не потеряется при перезагрузке
